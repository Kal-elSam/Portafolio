# UI Design Skill

Fuente de verdad:

- `docs/ai/ui.md`
- `docs/skills/ui-design.md`

Usar cuando:

- se crean o modifican componentes
- se trabaja con layouts, MDX o wireframes
- se ajusta tema o accesibilidad

Procedimiento:

1. Leer `docs/ai/ui.md`.
2. Reutilizar componentes en `apps/devSam/src/components/`.
3. Usar Tailwind + clsx, sin colores hardcodeados.
4. Verificar accesibilidad y mobile-first.
