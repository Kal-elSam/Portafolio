# Architecture Skill

Fuente de verdad:

- `docs/ai/architecture.md`
- `docs/skills/architecture.md`

Usar cuando:

- se añaden capas o módulos
- se evalúan refactors estructurales
- se necesita un ADR
- se introducen dependencias

Procedimiento:

1. Leer fuente de verdad.
2. Identificar capas afectadas.
3. Documentar ADR si aplica.
4. Validar contratos públicos.
5. Reportar riesgos.
