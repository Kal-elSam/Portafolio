# Conventions Skill

Fuente de verdad:

- `docs/ai/conventions.md`

Usar cuando:

- se genera código nuevo
- se revisan imports o naming
- se añaden archivos a `apps/devSam/src/`

Procedimiento:

1. Leer `docs/ai/conventions.md`.
2. Aplicar alias `@/` y orden de imports ESLint.
3. Respetar PascalCase en componentes.
4. No usar `any` sin justificación.
