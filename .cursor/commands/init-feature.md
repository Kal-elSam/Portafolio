# /init-feature

Inicializa una feature usando SDD + TDD.

## Flujo

1. Crear spec en `docs/ai/specs/<feature-name>.spec.md`.
2. Identificar capas afectadas (UI, API, lib, prisma).
3. Crear plan técnico.
4. Definir tests necesarios.
5. Crear estructura base si aplica.
6. No implementar hasta que la spec esté clara.
